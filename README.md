# Система распознавания лиц в реальном времени

Проект представляет собой систему распознавания лиц в реальном времени, созданную на Python с использованием технологий компьютерного зрения. Программа обнаруживает и идентифицирует лица в видеопотоке с веб-камеры, сравнивая их с изображениями известных лиц, сохранённых в системе.

---

## О проекте

В проекте применяются методы компьютерного зрения и распознавания лиц для идентификации людей в реальном времени. Система захватывает видео с камеры, находит лица и сопоставляет их с ранее сохранёнными изображениями, чтобы определить личность.

Области применения:

* Системы учёта рабочего времени
* Системы безопасности
* Контроль доступа
* Подтверждение личности

---

## Возможности

*  Обнаружение лиц в реальном времени с веб-камеры
*  Распознавание лиц на основе базы известных изображений
*  Хранение известных лиц в отдельной папке
*  Быстрая и эффективная обработка

## Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/a1beekovna/real_time_face_recognition_system.git
cd real_time_face_recognition_system
```

---

### 2. Установите зависимости

```bash
pip install opencv-python
pip install face-recognition
pip install numpy
```

Или установите всё сразу:

```bash
pip install opencv-python face-recognition numpy
```

---

## Запуск

Запустите основной скрипт:

```bash
python faceid3.py
```

или

```bash
python faceidWithstat.py
```

Откроется окно с веб-камерой, и система начнёт обнаруживать лица в реальном времени.

---

## Как добавить новые лица

1. Поместите изображение в папку:

```
known_faces/
```

Например:

```
known_faces/ivan.jpg
known_faces/maria.jpg
```

2. Запустите программу снова
3. Система будет распознавать нового человека

---

## Принцип работы

1. Загрузка изображений известных лиц из папки
2. Преобразование лиц в числовые векторы (кодировки)
3. Захват видео с веб-камеры
4. Обнаружение лиц в каждом кадре
5. Сравнение найденных лиц с известными кодировками
6. Вывод имени, если найдено совпадение
